# 역운49 비즈니스 모델 (BM)

> 최종 업데이트: 2026-02-24

---

## 1. 서비스 개요

**역운49**은 한국 전통 사주팔자(四柱八字)를 게이미피케이션으로 재해석한 운세 앱입니다.
사용자의 사주를 '초기 스탯'으로 삼아, 49일 동안 오행 기반 일일 퀘스트를 수행하며
다가오는 액운을 막고 운명을 개척하는 경험을 제공합니다.

### 핵심 가치
- **전통 × 게임**: 사주팔자라는 전통 콘텐츠를 RPG 퀘스트 시스템으로 재구성
- **AI 개인화**: Gemini AI를 활용한 맞춤형 사주 풀이
- **49일 사이클**: 불교의 49재 개념에서 영감을 받은 기간 한정 도전
- **수호신 시스템**: 오행 기반 캐릭터 수집 + 육성 요소

### 타겟 사용자
| 세그먼트 | 특징 |
|----------|------|
| 1차 타겟 | 20~40대, 운세/사주에 관심 있는 MZ세대 |
| 2차 타겟 | 자기계발 + 루틴 관리에 관심 있는 사용자 |
| 3차 타겟 | 캐릭터 수집형 게임을 즐기는 캐주얼 게이머 |

## 4. 수호신(조력자) 캐릭터 체계

### 기본 수호신 (무료, 5체)

| 이름 | 오행 | 모티브 | 설명 |
|------|------|--------|------|
| 청명 (淸明) | 수(水) | 해태 + 물의 정령 | 깊은 바다의 지혜, 감정의 균형 |
| 염화 (炎火) | 화(火) | 불삽살개 + 불의 정령 | 뜨거운 열정, 액운을 불태움 |
| 태풍 (颱風) | 목(木) | 아기 호랑이 + 바람 | 자유로운 바람, 새 생명력 |
| 무쇠 (武釗) | 금(金) | 현무/거북이 + 바위 | 철벽 방어, 묵직한 의지 |
| 황토 (黃土) | 토(土) | 아기 곰 + 땅의 정령 | 따뜻한 대지, 복과 안정 |

### 프리미엄 수호신 (유료, 향후 확장)

| 이름 (안) | 오행 | 특징 | 차별점 |
|----------|------|------|--------|
| (미정) | 수(水) 상위 | 용/이무기 모티브 | 특수 스킬: 흉 게이지 감소 효과 2배 |
| (미정) | 화(火) 상위 | 봉황/주작 모티브 | 특수 스킬: 퀘스트 보상 포인트 1.5배 |
| (미정) | 목(木) 상위 | 기린 모티브 | 특수 스킬: 특별 미션 추가 1개 |
| (미정) | 금(金) 상위 | 백호 모티브 | 특수 스킬: 재물운 관련 퀘스트 추가 |
| (미정) | 토(土) 상위 | 삼족오 모티브 | 특수 스킬: 일일 축복 확률 2배 |

> 프리미엄 수호신은 기본 수호신 대비 고유 스킬 + 고급 외형 + 특수 애니메이션 제공

## 9. 경쟁 분석

| 앱 | 특징 | 역운49 차별점 |
|----|------|-------------|
| 점신 | 일일 운세 + 타로 | 게이미피케이션 부재 → 49일 퀘스트로 차별화 |
| 사주팔자 | 사주 분석 전문 | 정적 해석만 제공 → AI 개인화 + 인터랙티브 |
| 운세 달력 앱 | 달력 기반 운세 | 참여형 콘텐츠 없음 → 캐릭터 + 미션 시스템 |
| 포춘 앱 | 서양 점성술 | 한국 전통 사주에 특화된 포지셔닝 |

### 역운49만의 핵심 차별점
1. **게이미피케이션**: 사주 = 읽기 전용 → 사주 = 플레이 가능한 RPG
2. **캐릭터 시스템**: 오행 기반 수호신 수집/육성 요소
3. **49일 한정 도전**: 기간 제한으로 긴장감 + 달성감 부여
4. **AI 개인화**: Gemini 기반 맞춤형 풀이 + 채팅 상담
